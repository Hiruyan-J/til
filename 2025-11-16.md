**【課題を通して学んだこと】**（知識・タスク分解などの振り返り）
- 条件付き代入演算子
	- ```
	a ||= b
	```
	aが未定義またはfalse/nilの場合にbがaに代入される
- executeメソッドの挙動
	- 戻り値は2次元配列が返される
	- SQLの集計関数(SUMなど)で計算した際に計算結果の値だけを取り出したい場合は、インデックス指定して1行1列目の配列の値を取り出す必要がある。
	```
	@total_expense = db.execute("SELECT SUM(amount) FROM account_books WHERE inout_type = 1")[0][0]
	```
	の[0][0]の部分は、
	```
	array = [[1, 2], [3, 4]]
	puts array[0][1] # => 2
	```
	とやっていることは同じ。
