- 範囲オブジェクト
    
    ```ruby
    (1..5).each do |i|
      puts "#{i}回目"
    end
    ```
    
    の `(1..5)` について調べてみた。範囲オブジェクトというらしい
    
    - `(1..5)` …「1から5までの範囲」を表すリテラル（特殊な書き方）
    - 範囲オブジェクトの作成方法
        - **`..`（ドット2つ）**: 範囲に**終了値を含む**（例: `(1..5)` は 1, 2, 3, 4, 5）。
        - **`...`（ドット3つ）**: 範囲に**終了値を含まない**（例: `(1...5)` は 1, 2, 3, 4）。
    - 主な用途
        - **繰り返し処理**（例: `(1..5).each`）
        - **条件分岐**（例: `if (1..5).include?(x)`）
        - **配列生成**（例: `(1..5).to_a` は `[1, 2, 3, 4, 5]` を生成）
- 修飾子
    
    
    - コードをシンプルに書く書き方
    - 例…if文
        
        ```ruby
        puts "成人です" if age >= 20
        ```
        
- メソッドの戻り値について
    - メソッド内で最後に記載された値が戻り値になる。    
    - そのため、わざわざ `return` を書かなくてもよい
