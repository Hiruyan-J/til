# 8/17(日)
**【学習時間】**
6.6h(累計:902h/1,000h)
  
**【学習内容】**
- Java
- RSpec
  

**【課題を通して学んだこと】**（知識・タスク分解などの振り返り）  
- Javaの特徴
    - Javaの主な特徴は2つ
        - オブジェクト指向
        - プラットフォーム非依存
            - JVMによってOSの違いを吸収してくれる
            Java仮想マシンのこと
            - JDK
                - Javaの開発環境
                - JDKとして含まれているもの
                    - コンパイラ、JVM、その他ツール
- Javaの開発の流れ
    - ソースプログラムの作成
        - `○○.java`
    - コンパイル
        - コマンドプロンプトで `javac ○○.java` でコンパイル
        - `○○.class` が生成される
    - 実行
        - `java クラス名` (拡張子は入力しない)
- プログラムの構成
    - Javaのプログラムは、**クラス**の集まり
    - クラス名とファイル名は合わせること
    - クラスの内容は`{ }` 間(ブロック)に書く
    - 処理のまとまりは**メソッド**
    - **mainメソッドから実行**される
    - 処理文は`;`で終わらせる
    - 書き方
        
        ```java
        class クラス名{
        	public static void main(String args[]){
        		処理文
        	}
        }
        ```
        
- 変数
    - 変数宣言
        
        ```java
        int num;
        ```
        
    - 複数同時に宣言
        
        ```java
        int month, day;
        ```
        
    - 変数宣言時に初期化
        
        ```java
        int age = 20;
        ```
        
- データ型
    - データ型は3,000種類
        - 基本データ型は8種類
            - intやlong、flootやchar、booleanなど
        - それ以外は、参照型
            - Stringなど
            - 変数には、データ自体のアドレス値(参照値)だけを格納している。
- コマンドラインからデータを受け取る方法
    - javaコマンド実行時に、クラス名の後ろで引数を渡す
    
    ```java
    java クラス名 Hello Java
    ```
    
- switch文
    - 注意点
        - `braak;` が無い場合、呼ばれた処理文以下の処理文が実行される(`default:`以外)
